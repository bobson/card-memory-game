# Memory Game

A simple card memory game implemented using web technologies.

## Description

Test your memory with this classic card matching game. Players need to find matching pairs of cards by flipping them two at a time.

## Features

- Interactive card flipping
- Score tracking
- Timer functionality
- Responsive design

## How to Play

1. Click on any card to reveal it
2. Click on another card to find a match
3. If cards match, they stay face up
4. If cards don't match, they flip face down
5. Continue until all pairs are matched

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/memory-game.git

# Navigate to project directory
cd memory-game

# Open index.html in your browser
```
